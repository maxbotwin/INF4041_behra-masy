Readme :

Application "PSG" qui permet à l'utilisateur de regarder la liste des 11 joueurs type du PSG.
Elle est alimentée par un fichier JSON que nous avons écrit nous même. 
Lorsqu'on clique sur un joueur on a quelques détails et un boutton qui permet de rechercher le joueur sur internet.

Une notification est présente dans toute l'application et mene à la page à propos.
On peut acceder a la page à propos par le menu également.
